## 2026-05-26 关键 Bug 修复

用户反馈：登录后所有功能不可用，任何操作都退回登录页。

**根因分析**：`db.put()` 新建记录时 `store.add()` 只返回自增整数 ID，不返回完整对象。导致：
- 首次登录后 `state.currentUser` 变成数字(如 `123`)，而非用户对象
- 所有代码中 `user.id`、`user.nickname` 访问均为 `undefined`
- 所有数据查询（按 `userId`）失败，页面静默崩溃

**修复内容（4个文件）**：
1. `db.js`: `put()` 新建记录返回 `{...data, id: req.result}`
2. `state.js`: `init()` 校验 user 对象结构，损坏数据自动清除
3. `router.js`: `navigate()` 增加显式 auth guard；简化 init 逻辑
4. `app.js`: `_doLogin()` 加 try-catch + 对象校验；`_saveCard()` 使用返回的完整对象；`init()`/`onNavigated()` 加错误隔离
